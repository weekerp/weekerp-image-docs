# 사이즈 조절

### 사이즈 조절 (Size Options)

이미지의 크기를 변경할 때는 `size`, `width`, `height` 파라미터를 사용합니다.\
`size`는 두 값을 한 번에 지정하는 **편의 옵션**이며,\
`width`·`height`는 각각 독립적으로 크기를 제어합니다.



#### `size(s)`&#x20;

**설명**\
가로(`width`)와 세로(`height`)를 동시에 설정할 수 있는 단일 옵션입니다.\
내부적으로 `width`/`height`로 확장되어 처리됩니다.

**형식**

```
?size={width}x{height}
```

**예시**

| 요청              | 결과               |
| --------------- | ---------------- |
| `?size=800x600` | 800×600 크기로 리사이즈 |

{% hint style="info" %}
이후 `width` 또는 `height` 파라미터가 존재하면  `size` 파라미터는 무시됩니다.
{% endhint %}



#### `width` (`w`)

**설명**\
출력 이미지의 가로(px)를 직접 지정합니다. 세로 값(`height`)이 없으면 **원본 비율 유지**로 자동 계산됩니다.

**예시**

| 요청                    | 결과                 |
| --------------------- | ------------------ |
| `?w=800`              | 가로 800px, 세로 자동 계산 |
| `?size=600x600&w=400` | width=400으로 덮어쓰기   |



#### `height` (`h`)

**설명**\
출력 이미지의 세로(px)를 직접 지정합니다.\
가로 값(`width`)이 없으면 **원본 비율 유지**로 자동 계산됩니다.

**예시**

| 요청                    | 결과                 |
| --------------------- | ------------------ |
| `?h=400`              | 세로 400px, 가로 자동 계산 |
| `?size=800x600&h=300` | height=300으로 덮어쓰기  |

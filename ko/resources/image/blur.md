# 블러 (모자이크)

### 블러 (Blur)

이미지에 **가우시안 블러(흐림 효과)** 를 적용할 수 있습니다.\
미리보기(placeholder) 연출, 민감 정보 가리기, 배경 처리 등에 유용합니다.

***

### `blur`

**설명**\
가우시안 블러의 강도(sigma)를 지정합니다. 값이 클수록 더 흐려집니다.



**형식**

```
?blur={sigma}
```



**지원 값**

* 숫자 값 (sigma)
* 지원 범위: `0.3 ~ 20`



**예시**

| 요청                              | 결과                        |
| ------------------------------- | ------------------------- |
| `?blur=1.5`                     | 약하게 블러 처리                 |
| `?width=800&format=webp&blur=2` | 800px 리사이즈 + WebP 변환 + 블러 |

***

#### 함께 사용하기

* `width`와 같이 쓰면 “가벼운 블러 썸네일(프리뷰)” 만들 때 효율이 좋아요.&#x20;
  * 예: `?width=80&blur=8` (저해상도 + 강블러 조합)

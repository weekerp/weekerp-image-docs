# 회전

### 회전 (Rotate)

이미지를 원하는 각도(degree)만큼 회전시킬 수 있습니다. 간단한 방향 보정, 연출, 썸네일 변형 등에 사용할 수 있습니다. 특히 회전되어 저장되는모바일 사진에서 유용합니다.

***

#### `rotate`

**설명**\
이미지를 지정한 각도만큼 회전합니다. 양수(+) 값은 **시계 방향**으로 회전합니다.



**형식**

```
?rotate={degrees}
```

#### 지원 값

* `-360 ~ 360`&#x20;



**예시**

| 요청             | 결과                 |
| -------------- | ------------------ |
| `?rotate=90`   | 시계 방향으로 90도 회전     |
| `?rotate=-323` | 시계 반대 방향으로 323도 회전 |

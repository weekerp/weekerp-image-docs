# 감마 보정 (이미지 중앙 톤)

### 감마 보정 (Gamma)

이미지의 **밝기 분포(중간톤)** 를 조정해서 화면에서 보이는 느낌을 바꿀 수 있습니다.\
단순히 “밝게/어둡게”가 아니라, **중간톤 중심으로** 톤이 변해서 사진/제품컷 보정에 자주 사용합니다.

***

### `gamma`



**설명**\
감마 보정 값을 지정합니다.\
보통 `1.8 ~ 2.2` 근처가 일반적으로 많이 쓰이는 범위입니다.



**형식**

```
?gamma={value}
```



**지원 값**

* 숫자 값
* 지원 범위: `1.0 ~ 3.0` &#x20;



**예시**

| 요청           | 결과            |
| ------------ | ------------- |
| `?gamma=2.2` | 중간톤 중심으로 톤 보정 |

***

### 함께 사용하기

* `format`, `quality`, `width` 같은 최적화 옵션과 함께 써도 자연스럽게 동작합니다.\
  예: `?width=1200&format=webp&quality=80&gamma=2.0`

***

{% hint style="info" %}
* 감마는 “전체 밝기”보다 **중간톤**을 변경하기에, 제품 사진에서 디테일을자연스럽게 보정할 때 유리합니다.
* &#x20;`1.8 → 2.0 → 2.2`처럼 **작게 단계적으로** 조정하면서 보는 게 안전합니다.
{% endhint %}

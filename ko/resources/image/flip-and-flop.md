# 플립 & 플롭

### 플립 (Flip)

이미지를 **상하/좌우로 뒤집는(반전)** 처리를 할 수 있습니다. 거울 효과, 셀카 방향 보정, 디자인 연출 등에 유용합니다.

### `flip`

**설명**

이미지를 **상하 반전**합니다. (위 ↔ 아래)



**형식**

```
?flip=true
```



**지원 값**

* `true | false`
* `true`일 때만 동작(미지정/false면 적용 없음)



**예시**

| 요청           | 결과          |
| ------------ | ----------- |
| `?flip=true` | 상하 반전(위↔아래) |



***

### 플롭 (Flop)

이미지를 **좌우로 뒤집는(반전)** 처리를 할 수 있습니다.&#x20;

### `flop`



**설명**

이미지를 **좌우 반전**합니다. (왼쪽 ↔ 오른쪽)



**형식**

`?flop=true`



**지원 값**

* `true | false`
* `true`일 때만 동작(미지정/false면 적용 없음)



**예시**

| 요청           | 결과         |
| ------------ | ---------- |
| `?flop=true` | 좌우 반전(왼↔오) |

#### 함께 사용하기

* `?flip=true&flop=true`
  * 상하 + 좌우를 모두 적용합니다. (결과적으로 180도 회전과 비슷한 효과가 나올 수 있음)



{% hint style="info" %}
* “셀카가 좌우가 뒤집혀 보이는 문제”는 `flop=true`로 해결됩니다. 썸네일 연출(거울 효과)은 flip/flop을 통해 유용하게 사용할 수 있습니다.
{% endhint %}

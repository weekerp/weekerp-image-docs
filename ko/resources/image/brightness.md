# 밝기 조절

### 밝기 조절 (Brightness)

이미지의 전체 밝기를 조정하여 더 밝거나 더 어두운 톤으로 보정할 수 있습니다.\
제품 사진, 배너, 썸네일 등에서 노출을 미세하게 맞추는 용도로 활용됩니다.

***

### `brightness`



**설명**\
이미지의 밝기 계수를 조정합니다.\
`1.0`은 원본과 동일하며, `1.0`보다 크면 더 밝아지고, 작으면 더 어두워집니다.



**형식**

```
?brightness={factor}
```



**지원 값**

* 지원 범위: `0.5 ~ 2.0`&#x20;



**예시**

| 요청                 | 결과        |
| ------------------ | --------- |
| `?brightness=1.15` | 밝기를 소폭 증가 |
| `?brightness=0.85` | 밝기를 소폭 감소 |

***

#### 함께 사용하기

* 밝기/색조(hue)와 함께 사용하여 톤을 더 정교하게 조정할 수 있습니다.\
  예: `?brightness=1.05&saturation=1.15`

***

{% hint style="info" %}
* 밝기 보정은 작은 폭(예: `1.05 ~ 1.15`)에서 차이가 충분히 느껴지는 경우가 많습니다.
* 과도한 값은 하이라이트가 날아가거나(너무 밝음) 디테일이 뭉개질 수 있으므로 단계적으로 조정하는 것을 권장합니다.
{% endhint %}

# 채도 조절

### 채도 조절 (Saturation)

이미지의 색감 강도를 조정하여 더 선명하거나 더 차분한 톤으로 보정할 수 있습니다.\
제품 사진, 음식 사진, 썸네일 등에서 색감을 일정하게 맞추는 용도로 활용됩니다.

***

### `saturation`



**설명**\
이미지의 채도 계수를 조정합니다.\
`1.0`은 원본과 동일하며, `1.0`보다 크면 채도가 증가하고, 작으면 채도가 감소합니다.



**형식**

```
?saturation={factor}
```



**지원 값**

* 지원 범위: `0.0 ~ 3.0` &#x20;



**예시**

| 요청                 | 결과        |
| ------------------ | --------- |
| `?saturation=1.25` | 채도를 소폭 증가 |
| `?saturation=0.85` | 채도를 소폭 감소 |

***

### 함께 사용하기

* 밝기/색조(hue)와 함께 사용하여 톤을 더 정교하게 조정할 수 있습니다.\
  예: `?brightness=1.05&saturation=1.15`

***

{% hint style="info" %}
* 채도는 과하게 올리면 피부톤이 부자연스럽거나 색 번짐이 커질 수 있어, `1.05 ~ 1.2` 범위에서 단계적으로 조정하는 것을 권장합니다.
* 채도를 낮추면 “차분한 무드”를 만들 수 있으며, 흑백(`grayscale`)과는 다르게 색 정보는 유지됩니다.
{% endhint %}

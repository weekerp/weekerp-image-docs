# 선명도 강화

### 선명도 강화 (Sharpen)

이미지의 가장자리(엣지)를 더 또렷하게 만들어 **선명도**를 올릴 수 있습니다.\
리사이즈 후 흐려 보이는 이미지를 보정하거나, 텍스트/로고를 또렷하게 보이게 할 때 유용합니다.

***

### `sharpen`

**설명**\
선명도 강도(sigma)를 지정합니다.\
값이 클수록 선명도 효과가 강해지지만, 이미지에 따라 **노이즈(거칠음)** 가 함께 증가할 수 있습니다.



**형식**

```
?sharpen={sigma}
```



**지원 값**

* `0.3 ~ 20`&#x20;



**예시**

| 요청             | 결과                     |
| -------------- | ---------------------- |
| `?sharpen=1.2` | 약하게 선명도 강화             |
| `?sharpen=3`   | 더 강한 선명도 강화(노이즈 증가 가능) |

***

#### 함께 사용하기

* `width`로 축소한 뒤 `sharpen`을 주면, 축소 과정에서 생기는 “살짝 뭉개짐”이 개선되는 경우가 많습니다.\
  예: `?width=800&sharpen=1.0`

***

{% hint style="info" %}
* 시작은 `1.0 ~ 1.5` 정도가 무난하고, 로고/텍스트는 `2.0` 근처에서 차이가 잘 느껴집니다.
* 피부/그라데이션처럼 부드러운 영역이 많은 사진은 sharpen을 올리면 **거칠어 보일 수** 있으니 낮은 값부터 올리는 게 안전합니다.
{% endhint %}

# 색상 조절

### 색조 조절 (Hue)

이미지의 색상(Hue)을 각도(degree) 기준으로 회전하여 전체적인 색조를 변경할 수 있습니다.\
일관된 무드 연출, 색감 보정, 특정 톤(따뜻함/차가움) 방향 조정 등에 활용됩니다.

***

### `hue`

**설명**\
색상(Hue)을 각도(degree) 기준으로 회전합니다. 양수(+) 값은 색상 휠을 **시계 방향**으로 회전합니다.

{% hint style="info" %}
Hue는 색상을 원형으로 배열한 **색상환(Color Wheel)** 기준으로 동작합니다.\
예를 들어 색상환은 대략 아래처럼 이어져요.

* 빨강 → 노랑 → 초록 → 하늘/청록 → 파랑 → 보라 → (다시) 빨강
{% endhint %}

`hue` 값은 이 색상환에서 **몇 도(degree)만큼 색을 이동**시키는 옵션입니다.



즉, `hue=30`이면 “현재 색을 색상환에서 30도만큼 다음 색 쪽으로 밀기”라고 이해할 수 있습니다.\
그래서 초록 잔디는 조금 더 노란 기가 돌거나, 파란 하늘이 청록 쪽으로 이동하는 식의 변화가 생깁니다.



**형식**

```
?hue={degrees}
```



**지원 값**

* 지원 범위: `-360 ~ 360`



**예시**

| 요청         | 결과          |
| ---------- | ----------- |
| `?hue=30`  | 색조를 30도 회전  |
| `?hue=-45` | 색조를 -45도 회전 |

***

### 함께 사용하기

* `saturation`, `brightness`와 함께 사용하면 “색 계열 이동 + 선명도/밝기”를 동시에 조정할 수 있습니다.\
  예: `?brightness=1.05&saturation=1.1&hue=10`

***

{% hint style="info" %}
* `hue`는 변화 폭이 큰 편이라, 먼저 `5~15` 정도의 작은 값으로 테스트하는 것을 권장합니다.
* 인물 사진은 피부톤이 쉽게 어색해질 수 있으므로, `hue`를 크게 돌리기보다는 `saturation/brightness`로 먼저 보정하는 편이 안정적입니다.
{% endhint %}

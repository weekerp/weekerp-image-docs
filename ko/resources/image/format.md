# 포맷 변환

### 포맷 변환 (Format & Quality)

Weekerp CDN은 이미지 요청 시점에\
**출력 포맷을 변경하거나 압축 품질을 조정**할 수 있습니다.\
이를 통해 트래픽 절감, 로딩 속도 개선, 브라우저 호환성 확보가 가능합니다.

***

#### `format`



**설명**\
출력할 이미지 포맷을 지정합니다.\
지정하지 않으면 원본 포맷으로 반환됩니다.



**형식**

```
?format={type}
```



**별칭**

* `f` (짧은 형태)
* `format` (정식 형태)



**지원 포맷**

| 포맷            | 설명         | 특징              |
| ------------- | ---------- | --------------- |
| `webp`        | 고효율 압축 포맷  | 대부분의 현대 브라우저 지원 |
| `avif`        | 차세대 고효율 포맷 | WebP보다 더 높은 압축률 |
| `jpeg`, `jpg` | 일반적인 손실 포맷 | 균형 잡힌 호환성       |
| `png`         | 무손실 압축 포맷  | 투명도 지원          |
| `gif`         | 애니메이션 이미지  | 단순 애니메이션에 적합    |
| `tiff`        | 고품질 포맷     | 인쇄·보관용          |
| `svg`         | 벡터 포맷      | 로고, 아이콘용        |



**예시**

| 요청             | 결과       |
| -------------- | -------- |
| `?format=webp` | WebP로 변환 |
| `?format=avif` | AVIF로 변환 |
| `?f=png`       | PNG로 변환  |

***

#### &#x20;`quality`



**설명**\
압축 포맷의 이미지 품질을 조정합니다.\
값이 낮을수록 용량은 줄지만 세부 품질이 감소합니다.



**형식**

```
?quality={1~100}
```



**별칭**

* `q`



**예시**

| 요청      | 결과             |
| ------- | -------------- |
| `?q=90` | 고품질 (용량↑)      |
| `?q=60` | 중간 품질 (기본값 수준) |
| `?q=30` | 저품질 (용량↓)      |

***

#### 함께 사용하기

```txt
?format=webp&q=75
```

* AVIF·WebP·JPEG 등 손실 포맷에 적용됨
* PNG·SVG는 무손실 포맷이므로 `quality`는 무시됨

***

#### MIME 자동 변환

요청 시 `format`에 따라 응답 헤더의 `Content-Type`이 자동 변경됩니다.

| format         | Content-Type    |
| -------------- | --------------- |
| `webp`         | `image/webp`    |
| `avif`         | `image/avif`    |
| `jpeg` / `jpg` | `image/jpeg`    |
| `png`          | `image/png`     |
| `svg`          | `image/svg+xml` |

***

> 💡 **Tip:**
>
> * WebP는 대부분의 브라우저에서 기본 지원되며, JPEG 대비 용량을 25\~35% 절감할 수 있습니다.
> * AVIF는 고효율이지만, 일부 구형 브라우저에서는 호환되지 않을 수 있습니다.

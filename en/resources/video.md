# Video API

We are currently expanding features in stages.\
If you need testing access or early integration, please contact us at **help@weekerp.com**.\
Beta access is being provided gradually.

At this stage, you can use the **`/file`** endpoint instead of **`/video`**.

**Example:**

```
cdn.weekerp.com/file/example.mp4
```
